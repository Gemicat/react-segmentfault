# 使用React抓取segmentfault得文章及信息


> 使用方法

首先，clone本项目到本地

    git clone https://github.com/Gemicat/react-segmentfault.git

进入项目路径

    cd react-segmentfault

然后，安装node相关依赖，开启服务

    npm install
    node server.js

接着进入到项目目录下的app目录，安装依赖并启用webpak-dev-server

    npm install
    npm start

> 使用前必须安装有全局的node环境和webpack环境
